# 이서진 포트폴리오 사이트 운영 사용설명서

이 문서는 HTML, CSS, JavaScript, GitHub를 모르는 사람도 포트폴리오 내용을 수정할 수 있도록 만든 안내서입니다.

사이트 주소: <https://b35436656-stack.github.io/Lee-seojin-portfolio/>

## 1. 운영 방식

사이트 디자인과 기본 구조는 GitHub Pages에 있습니다.

앞으로 내용 수정은 아래 두 곳에서 합니다.

- 글자 내용 수정: Google Sheets
- 사진, 이미지, 이력서 PDF 교체: Google Drive

Google Sheets에 적은 내용은 사이트가 자동으로 읽어옵니다. Google Drive 파일은 공유 링크를 Google Sheets에 붙여넣으면 사이트에서 표시됩니다.

## 2. 처음 한 번만 준비할 것

### Google Sheets 만들기

1. Google Drive에 접속합니다.
2. 새로 만들기 > Google 스프레드시트를 선택합니다.
3. 파일 이름을 `이서진 포트폴리오 관리표`로 바꿉니다.
4. 시트 탭을 아래 3개로 만듭니다.

- `Settings`
- `Experience`
- `Portfolio`

### 템플릿 붙여넣기

이 폴더의 `google-sheets-template` 안에 있는 CSV 파일 3개를 열어 각 시트에 붙여넣습니다.

- `Settings.csv` 내용은 `Settings` 시트에 붙여넣기
- `Experience.csv` 내용은 `Experience` 시트에 붙여넣기
- `Portfolio.csv` 내용은 `Portfolio` 시트에 붙여넣기

### Google Sheets 공개 설정

1. Google Sheets 상단 메뉴에서 파일 > 공유 > 웹에 게시를 누릅니다.
2. 전체 문서를 웹에 게시합니다.
3. 공유 버튼에서도 링크가 있는 모든 사용자가 볼 수 있도록 설정합니다.
4. 주소창의 URL에서 `/d/`와 `/edit` 사이의 긴 문자를 복사합니다.

예시:

```text
https://docs.google.com/spreadsheets/d/여기가_SHEET_ID입니다/edit
```

복사한 값을 `site-config.js` 파일의 `googleSheetId`에 한 번만 넣으면 됩니다.

```js
window.PORTFOLIO_CONFIG = {
  googleSheetId: "여기에 Google Sheet ID 입력",
  cacheMinutes: 5
};
```

이 작업은 처음 연결할 때만 필요합니다. 이후에는 Google Sheets만 수정하면 됩니다.

## 3. 자기소개 수정 방법

Google Sheets의 `Settings` 시트에서 아래 항목의 `value` 칸을 수정합니다.

- `heroTitle`: 첫 화면 큰 제목
- `heroSubtitle`: 첫 화면 설명 문장
- `aboutTitle`: About 페이지 제목
- `aboutIntro`: About 페이지 소개 문장
- `resumeTitle`: Resume 페이지 제목
- `resumeSubtitle`: Resume 페이지 설명 문장

줄을 나누고 싶으면 셀 안에서 줄바꿈을 넣으면 됩니다.

## 4. 경력 추가 방법

Google Sheets의 `Experience` 시트에서 새 줄을 추가합니다.

각 칸의 의미는 아래와 같습니다.

- `period`: 기간, 예: `2022.02 - 2023.12`
- `company`: 회사명 또는 기관명
- `role`: 맡은 역할
- `summary`: 경력 요약 설명
- `details`: 세부 업무. 여러 줄로 적으면 목록으로 보입니다.
- `tags`: 키워드. 쉼표로 구분합니다.
- `visible`: 사이트에 보이게 하려면 `TRUE`, 숨기려면 `FALSE`

경력을 삭제하고 싶으면 행을 지우기보다 `visible`을 `FALSE`로 바꾸는 것이 안전합니다.

## 5. 포트폴리오 추가 방법

Google Sheets의 `Portfolio` 시트에서 새 줄을 추가합니다.

각 칸의 의미는 아래와 같습니다.

- `category`: 사례 분류
- `title`: 포트폴리오 제목
- `summary`: 짧은 설명
- `problem`: 어떤 문제가 있었는지
- `action`: 내가 무엇을 했는지
- `result`: 의미, 결과, 배운 점
- `tags`: 키워드. 쉼표로 구분합니다.
- `imageUrl`: Google Drive 이미지 공유 링크
- `imageAlt`: 이미지 설명
- `visible`: 사이트에 보이게 하려면 `TRUE`, 숨기려면 `FALSE`

자동화 코드처럼 보호해야 하는 자료는 코드 전문을 올리지 말고, 문제와 시도 과정만 설명하세요.

## 6. 사진 교체 방법

1. Google Drive에 사진 파일을 올립니다.
2. 파일을 우클릭하고 공유를 누릅니다.
3. 링크가 있는 모든 사용자가 볼 수 있도록 바꿉니다.
4. 링크 복사를 누릅니다.
5. Google Sheets에 붙여넣습니다.

프로필 사진은 `Settings` 시트의 `profileImageUrl`에 넣습니다.

포트폴리오별 사진은 `Portfolio` 시트의 `imageUrl`에 넣습니다.

## 7. 이력서 PDF 교체 방법

1. Google Drive에 새 이력서 PDF를 올립니다.
2. 파일을 우클릭하고 공유를 누릅니다.
3. 링크가 있는 모든 사용자가 볼 수 있도록 바꿉니다.
4. 링크 복사를 누릅니다.
5. `Settings` 시트의 `resumePdfUrl`에 붙여넣습니다.

잡코리아 이력서 PDF가 따로 있으면 `jobkoreaPdfUrl`에 넣습니다.

## 8. 연락처 수정 방법

Google Sheets의 `Settings` 시트에서 아래 값을 수정합니다.

- `contactEmail`: 이메일
- `contactPhone`: 전화번호

사이트에서는 연락처가 바로 노출되지 않고, Contact 페이지의 `연락하기` 버튼을 눌렀을 때 표시됩니다.

## 9. 변경사항 반영 시간

보통 Google Sheets를 수정한 뒤 1분에서 5분 안에 반영됩니다.

사이트는 너무 자주 Google Sheets를 불러오지 않도록 약 5분 단위로 새 내용을 확인합니다.

바로 확인하고 싶으면 브라우저에서 새로고침을 누르세요.

## 10. 문제가 발생했을 때 확인 방법

### 내용이 바뀌지 않을 때

1. Google Sheets의 시트 이름이 정확한지 확인합니다.
   - `Settings`
   - `Experience`
   - `Portfolio`
2. Google Sheets가 웹에 게시되어 있는지 확인합니다.
3. Google Sheets 공유 설정이 링크가 있는 모든 사용자 보기로 되어 있는지 확인합니다.
4. 5분 정도 기다린 뒤 새로고침합니다.

### 사진이 안 보일 때

1. Google Drive 사진 공유 설정을 확인합니다.
2. 링크가 있는 모든 사용자가 볼 수 있어야 합니다.
3. Google Sheets에 사진 파일 링크가 제대로 들어갔는지 확인합니다.

### PDF가 안 열릴 때

1. Google Drive PDF 공유 설정을 확인합니다.
2. 링크가 있는 모든 사용자가 볼 수 있어야 합니다.
3. `Settings` 시트의 `resumePdfUrl` 값이 새 PDF 링크인지 확인합니다.

### 일부 경력이나 포트폴리오가 안 보일 때

`visible` 칸이 `FALSE`이면 사이트에 보이지 않습니다. 보이게 하려면 `TRUE`로 바꾸세요.

## 11. 안전하게 운영하는 법

- 주민등록번호, 계좌번호, 계약서 원본, 카드결제 정보는 절대 올리지 마세요.
- 회사 내부자료는 원본 이미지나 문서 대신 요약 설명만 올리세요.
- 자동화 코드 전문은 올리지 마세요.
- 포트폴리오에는 문제, 내가 한 행동, 배운 점 중심으로 적는 것이 안전합니다.
