# 관리자 사용설명서

운영자는 Google Sheets와 Google Drive만 사용합니다.

## 기본 정보 수정

`Settings` 탭에서 `value` 칸을 수정합니다.

- `siteTitle`: 브라우저 제목
- `siteName`: 메뉴에 보이는 이름
- `personName`: 사람 이름
- `jobTitle`: 희망 직무
- `heroTitle`: 첫 화면 큰 제목
- `heroSubtitle`: 첫 화면 설명
- `aboutTitle`: 소개 페이지 제목
- `aboutIntro`: 소개 페이지 짧은 설명
- `aboutBody`: 자기소개 본문
- `footerText`: 하단 문구

줄바꿈은 셀 안에서 줄을 바꾸면 사이트에도 반영됩니다.

## 연락처 수정

`Settings` 탭에서 수정합니다.

- `contactEmail`
- `contactPhone`
- `snsLinkedIn`
- `snsGithub`
- `snsBlog`
- `snsInstagram`

SNS 링크는 비워두면 사이트에 표시되지 않습니다.

## 경력 추가

`Experience` 탭에서 새 행을 추가합니다.

- `sortOrder`: 숫자가 작을수록 위에 표시
- `period`: 기간
- `company`: 회사명
- `role`: 역할
- `summary`: 설명
- `details`: 세부 업무
- `tags`: 쉼표로 구분한 키워드
- `visible`: 보이게 하려면 `TRUE`, 숨기려면 `FALSE`

## 포트폴리오 추가

`Portfolio` 탭에서 새 행을 추가합니다.

- `sortOrder`: 숫자가 작을수록 위에 표시
- `category`: 분류
- `title`: 프로젝트명
- `summary`: 설명
- `techStack`: 기술스택 또는 사용 도구
- `projectUrl`: 관련 링크
- `imageUrl`: Google Drive 이미지 링크
- `imageAlt`: 이미지 설명
- `problem`: 문제
- `action`: 내가 한 행동
- `result`: 결과 또는 의미
- `tags`: 키워드
- `visible`: 보이게 하려면 `TRUE`, 숨기려면 `FALSE`

## 사진 교체

1. Google Drive에 사진을 업로드합니다.
2. 공유 설정을 `링크가 있는 모든 사용자`로 바꿉니다.
3. 링크를 복사합니다.
4. 프로필 사진은 `Settings` 탭의 `profileImageUrl`에 넣습니다.
5. 배경 이미지는 `backgroundImageUrl`에 넣습니다.
6. 프로젝트 이미지는 `Portfolio` 탭의 `imageUrl`에 넣습니다.

## 이력서 PDF 교체

1. Google Drive에 PDF를 업로드합니다.
2. 공유 설정을 `링크가 있는 모든 사용자`로 바꿉니다.
3. 링크를 복사합니다.
4. `Settings` 탭의 `resumePdfUrl`에 붙여넣습니다.

## 색상 변경

`Settings` 탭에서 아래 값을 바꿉니다.

- `mainColor`
- `subColor`
- `backgroundColor`
- `textColor`
- `buttonColor`
- `linkColor`
- `accentColor`
- `profileCardBg`
- `sectionBg`
- `footerBg`
- `profileAccentColor`

색상은 `#3e6b5b`처럼 입력합니다. 잘못 입력하면 기본 색상이 사용됩니다.

## 글자 크기 변경

`Settings` 탭에서 아래 값을 바꿉니다.

- `titleFontSize`: 예 `56px`
- `subtitleFontSize`: 예 `18px`
- `bodyFontSize`: 예 `16px`
- `menuFontSize`: 예 `15px`
- `buttonFontSize`: 예 `15px`

## 테마 변경

`siteTheme` 값에 아래 중 하나를 입력합니다.

- `Light`
- `Dark`
- `Ocean`
- `Forest`

## 텍스트 강조 문법

HTML을 몰라도 아래 문법을 사용할 수 있습니다.

```text
**굵은 글씨**
*기울임*
__밑줄__
[red]빨간색[/red]
[blue]파란색[/blue]
[green]초록색[/green]
[color:#ff6600]사용자 지정 색상[/color]
[link:https://example.com]링크 텍스트[/link]
```

사이트가 자동으로 서식으로 바꿔줍니다.
